在IIS6下安装PHP+MYSQL+PHPMYADMIN

发表于 2009-02-28 更新于 2019-07-25 分类于 网站建设

今天在win2003主机上安装了PHP+MYSQL+PHPMYADMIN,成功后,把心得记录下来,自己以后使用也方便些. 1,首先IIS6是2003主机上已经有的了,就不在叙述了. 2,安装php,我主要是参考www.php.net上的"[手动安装说明](http://cn2.php.net/manual/zh/install.windows.manual.php)",下载的是PHP手动解压包.我安装在D:/PHP目录了.进入d:/php目录,将php.ini-recommended文件改名为php.ini.然后打开php.ini, a,找到里面的

;extension=php_mysql.dll 将前面的

; 号去掉，

改成：extension=php_mysql.dll b,找到:extension_dir = “./”

将其改php 安装目录下

ext 子目录的绝对路径.我的是

extension_dir = “d:/php/ext/”

c,把php.ini复制到Windows目录下面，把libmysql.dll复制到Windows下面的System32目录. 3,我用的是CGI方式执行PHP脚本.所以我在IIS里是这样设置的:进入

web 服务扩展，点击“添加一个新的

web 服务扩展”，“扩展名”中输入名称，例如

PHP，再点击“添加”按钮，点击“浏览”

CGI 文件

php-cgi.exe作为“文件路径”后点“确定”，然后选中“设置扩展状态为允许”并点“确定”. 4,打开“网站”->“属性”->“文档”->“启用默认内容文档”->“添加”，可以将

index.php 添加为默认内容文档.当然也可以多添加几个… 5,在cmd里运行

net stop iisadmin (停止IIS服务) net start w3svc

(重新启动IIS服务) 6,在记事本中输入下面的代码：

< ?php phpinfo(); ? > 然后保存为index.php，放在网站的根目录，在浏览器中打开

http://localhost/index.php，即可看到测试输出结果. 安装MYSQL比较简单,下个MYSQL_WIN安装版就可以了.里面有几个步骤需要选择,看下它的说明,一路NEXT基本就OK了,记得建立ROOT的密码时候

第一个密码应该是空的,因为是第一建立,下面两个密码选项才是你现在应该设置的,当然要设置的复杂点了…. PHPMYADMIN直接上传到你虚拟空间的根目录,就可以配置config文件

打开libraries下的config.default.php文件，依次找到下面各项，按照说明配置即可：

A.访问网址

$cfg[‘PmaAbsoluteUri’]

= ‘’;这里填写phpmyadmin的访问网址

B.mysql主机信息

$cfg[‘Servers’][$i][‘host’]

= ‘localhost’; // MySQL hostname or IP address 填写localhost或mysql所在服务器的ip地址，如果mysql和该phpmyadmin在同一服务器，则按默认localhost $cfg[‘Servers’][$i][‘port’]

= ‘’; // MySQL port - leave blank for default port mysql端口，如果是默认3306，保留为空即可

C.mysql用户名和密码

$cfg[‘Servers’][$i][‘user’]

= ‘root’; // MySQL user 访问phpmyadmin使用的mysql用户名

fg[‘Servers’][$i][‘password’]

= ‘’; // MySQL password (only needed对应上述mysql用户名的密码

D.认证方法

$cfg[‘Servers’][$i][‘auth_type’]

= ‘cookie’; 在此有四种模式可供选择，cookie，http，HTTP，config config方式即输入phpmyadmin的访问网址即可直接进入，无需输入用户名和密码，是不安全的，不推荐使用。

当该项设置为cookie，http或HTTP时，登录phpmyadmin需要数据用户名和密码进行验证，,具体如下：

PHP安装模式为Apache，可以使用http和cookie；

PHP安装模式为CGI，可以使用cookie E.短语密码(blowfish_secret)的设置

$cfg[‘blowfish_secret’]

= ‘’; 如果认证方法设置为cookie，就需要设置短语密码，置于设置为什么密码，由您自己决定

，但是不能留空，否则会在登录phpmyadmin时提示错误

好了，到此为止，您已经成功安装了phpmyadmin 以上就是我今天的成果了,呵呵

,赶紧在主机空间里建了个基于Wordpress的BLOG程序,测试成功.
