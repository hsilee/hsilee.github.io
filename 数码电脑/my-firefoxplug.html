我的Firefox插件及简介

发表于 2009-03-09 更新于 2019-07-25 分类于 数码电脑

Firefox是一个自由的,开放源码的浏览器,适用于

Windows, Linux 和

MacOS X平台,它体积小速度快，还有其它一些高级特征,主要特性有:标签式浏览,使上网冲浪更快;可以禁止弹出式窗口;自定制工具栏;扩展管理;更好的搜索特

性;快速而方便的侧栏. 我之所以选择Firefox,最关键原因还是其强大的插件定制功能.下面我就简单介绍下我所使用的Firefox插件; 1, Adblock Plus

你喜欢广告吗?如果你喜欢,请千万别装它

!

(建议安装) 2,

BBCode

这个应该是小范围应用插件,

通过它可以使用右键菜单方便地添加

BBCode/HTML/XHTML 等格式代码，如文字大小、文字颜色、粗体、斜体、插入图像、表情符号、网址链接等等，为你在

Blogs 和论坛发言提供辅助. 3,

Easy DragToGo

轻松使用拖曳手势打开标签页,呵呵,这个插件你真的是用上了才知道有多方便.很多遨游的用户转到Firefox应该都会选用这个插件吧? (建议安装) 4,

FEBE 没啥可说的,备份你的Firefox用的.

(建议安装) 5, FLASHgot

通过这个插件,你可以调用下载工具(比如迅雷,快车),来下载东西.

(建议安装) 6, foxyproxy

这个是个代理浏览插件,需要配合TOR来使用

. 7, IE TAB 这个也是没啥说的

,可以让Firefox调用IE核心.如果你遇到

only ie的网站,只能通过这个插件才可以浏览网站

(建议安装) 8, media wrap

使

Firefox 也能够象

IE 一样正常播放

Wmplayer、Rmplayer、QtPlayer 和

FlashPlayer 格式的媒体文件。当然，你的系统中首先要安装过相应的播放软件及解码器，比如，Windows Media Player 和

RealPlayer 等

(建议安装) 9, New Tab Button Position

在标签栏选定的位置添加IE7风格的新建标签页按钮.呵呵,据说3.1以后,Firefox直接支持这个功能了. 10,

WebMail Notifier

这个插件可以检查

webmail 帐号并且通报未读邮件数量.而且通过添加脚本,也可以支持QQ,163等邮箱. (建议安装)
