bluehost空间申请免费SSH及搭建

发表于 2011-01-29 更新于 2019-07-27 分类于 网站建设

今天把前几天买的bluehost提供的SSH服务申请下来了.简单记录下,以备自己以后使用.

1,首先登录bluehost的cpanel管理面板,然后在help里搜索SSH.里面有个联系的链接,点进去. 大致的填写内容; “hi, Please enable SSH for my hosting account. The following is my account informaiton: primary domain: my-domain.com”

记得在附件位置上传一张自己的身份证照片. 然后把用户名和登录密码后4位填到指定的表格,还有你购买bluehost填写的email也一并填到指定位置,然后等待即可. 这里一定要表扬下bluehost,下面说明写的是在24-48小时内回复,实际时间5分钟.开通的邮件就发到我的邮箱里了.也可能是美国那边7点的原因吧.反正就是惊人的办事效率.

2,既然开通了,进去小小设置下,在cpanel面板-安全-SSH/壳访问

(点进去,生成自己的SSH公私密钥.这里设置很简单,起个名字弄个密码就完成了,然后把私人密钥下载到自己电脑即可)

3,SSH链接工具推荐用putty,毕竟是开源的,用着安全放心.网上还有一款中文界面的myentunnel.我没用:) putty配置简单说下,先在SSH目录下选择tunnels,目标设置为Dynamic,添加一个端口任意端口(比如8848 嘿嘿)，再按Add,一个动态转发端口就实现了. 然后用相应帐号ssh登录后,除了登录的终端窗口以外,刚才那个8848连服务器的22端口之间就有了一个SSH加密的转发通道了. 由于是ms-dos窗口

我就看到[~]就算是链接上了…. 有点简陋.不过下次再登录会有时间提示. 到这里SSH加密通道大功告成了.

4, 关于FTP工具上传,小小吐槽下啊.平时使用flashfxp顺手了,结果死活登录不进去…..后来老老实实用官方推荐的coreftp,好好享受了一把Sftp的快乐啊,速度确实比普通的速度快,而且还是加密的,呵呵

5,上面就是流水账一般把SSH链接简单记录了下,以下内容我就不负责了,比如某些喜欢”跳”的同学可以看看.我是google fans.这个不用总强调了,所以我当然用chrome了,chrome嘛,去下载一个叫
