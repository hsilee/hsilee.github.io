57个不为人所知却很有用的Google产品

必备

01.谷歌阅读器(Google Reader)：网页版RSS阅读器，方便订阅，组织和分享新闻。有手机版。

02.个性化主页(iGoogle)：显示Gmail，日历，天气，新闻等等，多模板。

03.谷歌相册服务(Google Picasa)：提供照片的下载和编辑软件，用这个软件Email，上传，张贴照片到博客，也可以直接使用网络相册。

04.谷歌文件(Google Docs)：您不再需要桌面版的Office了，这个是谷歌的在线Office，你可以创建，发布，共享和协作在线文档和幻灯。

05.谷歌记事本(Google Notebook)：剪贴文本，图片和链接页面，可以保存，检索或共享。

06.谷歌桌面(Google Desktop)：桌面搜索，可以搜索你电脑里的电子邮件，计算机文件，音乐，照片，聊天和Web历史记录，和iGoogle类似，也安装小工具。

地图和旅游

07.谷歌搭车(Google Ride Finde)r：交通搭乘查询服务，通过这个搜索工具，利用GPS数据钩，通过Google地图，在14个美国城市连接出租车，轿车和公共汽车的搭车搜索。

08.谷歌公交(Google Transit): 使用Google地图，告诉谷歌你在哪，要到哪，谷歌帮助你安排公共交通线路方案。

09.谷歌火星(Google Mars)：和谷歌地球类似，只是这个是火星而已。

10.谷歌天空(Google Sky)：在不夜城也可以看星星，你可以通过虚拟的太空望远镜，看行星，恒星，星系，卫星等等。

11.谷歌地球(Google Earth): 下载一个吧，你就什么都知道了。

网站浏览和开发

12.谷歌3D建模(Google SketchUp)：简单而有效的3D工具，Mac和PC都可以使用，建筑师，土木工程师，设计师，甚至是游戏开发商的方便工具。允许用户上传他们的模型到谷歌地球。

13.谷歌支付(Google Checkout): 谷歌的支付宝，目前中国没有开通。

14.谷歌网络加速器(Google Web Accelerator)：类似于CDN的加速服务，利用谷歌在全球的服务器，缓存网站，让你访问速度倍增。

15.FeedBurner：转烧你的Feed，便于订阅，不怕当机。

16.搜索历史(Google Web History)：记录你的网页浏览记录，以供你搜索。

17.谷歌Base：可以存储文字，图片文件和网页的数据库，搜索结果显示在网页搜索，谷歌地图和产品搜索。

18.自定义搜索引擎(Google Co-Op): 完善和分类搜索查询，创建定制的搜索引擎，搜寻专门信息。

19.谷歌应用引擎(Google App Engine)：

谷歌的云计算服务，相当于一种虚拟服务器，目前仅仅支持Python。

20.谷歌网站优化器(Google Website Optimizer)：检查您的网站，测试网站内容，帮助您创建最有效和最有利可图的广告，创造友好的界面，增加转换率。

21.谷歌浏览器同步(Google Browser Sync): 通过互联网，同步FireFox的密码，书签，历史，标签，窗口到另外的电脑。

22.谷歌点击电话(Google Click-to-Call): 这个是一个搜索结果页的广告系统，你可以免费打电话个做广告的公司，至少是谷歌付钱。你输入i的电话号码，你就可以和广告商通话。

23.谷歌页面生成器(Google Page Creator): 简单的个人网页，提供简单的网页编辑器，100M空间，多种模板，有点象傻瓜建站。

社会化网络和通讯

24.Orkut社区：很红的巴西和印度社区，美国不流行。

25.Android手机操作系统：基于Linux的手机操作系统。

26.发短信到收集(Google Send to Phone)：通过FireFox插件发短信给自己。

27.谷歌分享(Google Shared Stuff): 分享网页给好友，可以整合到Facebook和del.icio.us。

28.GTalk(Google Talk)：

谷歌的聊天软件，在Gmail里面也可以用。

29.Dodgeball手机社区：在美国的22个城市，你可以知道你的朋友在哪里，朋友也知道你，可以要他们来找你。

30.连接朋友(Google Friend Connect): 可以让用户方便地为网站添加社会化网络功能，就像Facebook和Orkut，无需很深的编程技术。

31.谷歌呼叫中心(Google GrandCentral)：

用户只要一个电话号码，换手机、搬家、换工作，还是可以继续用原来的号码。

个性搜索工具

33.谷歌学术搜索(Google Scholar): 面向学术资源的免费搜索工具,能够帮助用户查找包括期刊论文、学位论文、书籍、预印本、文摘和技术报告在内的学术文献,内容涵盖自然科学、人文科学、社会科学等多种学科。

35.谷歌预测搜索(Google Sets)：用户提交一个搜索关键字之后,它可以把相关的关键字列出来,为用户的搜索提供相关提示! 36.谷歌邮购搜索服务(Google Catalogs): 不用在往垃圾桶里扔废纸了。

38.谷歌盲人搜索(Google Accessible Search) 39.谷歌趋势(Google Trends) 40.谷歌图书搜索(Google Book Search) 41.谷歌新闻档案搜索(Google News Archive Search)：搜索老新闻。

42.谷歌特殊搜索(Google Special Searches)：搜索范围很小的搜索，比如政府，公共服务，微软，苹果等等。

44.谷歌小工具(Google Gadgets): 和iGoogle和Desktop配合使用。

谷歌的话务员人肉搜索，只对加拿大和美国开放。

也说"技术"足球

发表于 2008-06-20 更新于 2019-07-25 分类于 体育

现在无论是所谓的足球专家也好,评论员也好,球迷也好.看起球来都喜欢说个”技术”.足球本来就是一门上升的艺术的体育运动,既然上升到”艺术”的高度了.其中必须得有技术含量,也就是所谓的”技术足球”.但是问题出来了,现在大多数人眼里的”技术”就是花哨的盘带过人,仿佛这就是技术足球了.呵呵,对不起真的要笑下,试问如果只讲盘带,当今足坛也就是葡萄牙,西班牙类的拉丁足球算是技术了.巴西都算不上,但是有意思的是为什么最后的胜利不是这些所谓的”技术足球”取得胜利呢?你别光强调运气,身体什么的.就拿荷兰和德国来说吧,一般的球迷,包括很多足球的评论员张口就说技术不行,好听点的叫技术加力量.但是德国为什么能成为欧洲之王呢?别忘记欧洲杯的3星不是盖的,光靠意志和身体,就能取得这么骄人的战绩? 其实大家往往忽略了一个概念,其实穿插,跑位,抢截,特别是漂亮的一脚传球也都是技术.特别是现在足球的特点,各个球队都很注重防守.有时候漂亮的一脚传球远比花哨的盘带过人更实用.其实也更需要”技术”!最后我要说的就是,我不是否认盘带过人等花哨的动作,只是咱们看球还是不能光盯着一点来看.足球里的技术不单单是漂亮的盘带过人,还有很多表现形式,别光看一点来评价任何一支球队. 他们都很技术,也许是评球的您,不懂技术!

暴雪发布官方暗黑II免CD补丁!

发表于 2008-06-19 更新于 2019-07-25 分类于 游戏

暴雪的敬业态度确实值得每一家游戏厂商学习。今年1月中旬，暴雪为《星际争霸》和资料片《母巢之战》发布了1.15.2版补丁，去掉了游戏过程中的CD验证机制，也就相当于一个官方免CD补丁。

现在，“大菠萝”《暗黑破坏神2》及其资料片《破坏之王》也获得了同样的待遇，更新了今天的1.12a版补丁后也不需要CD了。经过这么多年的洗礼，相信不少玩家的光盘都快支撑不住了，所以说暴雪还是相当体贴的。

如果用户当初安装游戏的时候选择了完全安装，并且不希望在游戏的同时插入CD光盘，那么只需把光盘上的所有.MPQ文件复制到游戏目录即可，大部分用户只需要复制D2Music.mpq或者D2xMusic.mpq，但Mac用户复制这些音乐文件之后还需要重命名为“Diablo II Music”或者“Diablo II Expansion Music”。

对于执行了非完全安装的玩家，那就需要重装一次游戏，当然这次要选择完全安装了。

除了免CD，新版补丁还修正了在Intel Mac平台上Rosetta和OpenGL模式之间的兼容性问题。

看eq2东方版的倒闭---失败的"橘子"

发表于 2007-03-30 更新于 2019-07-25 分类于 游戏

前几天在网上看到eq2东方版关闭运营的消息.并没有感觉很惊讶,因为在所有关心并喜欢这个游戏的广大fans心中都有数的.相信大多数fans都预感到这个结局了.但是为什么这么好的一款游戏,或者类似的几款优秀游戏都在大陆折戟沉沙,屡遭败绩呢?比如eq1,2. 魔剑.等等….. 不能单说一句水土不服就能解释得了的吧?因为在中间还伴随着魔兽世界(wow)这个成功的例子.本人觉得别的游戏不说,单说eq2在中国的几个失误地方

1,东方版本身就是失败.为什么要搞个东方版?这个想法我一直很不理解.游戏橘子难道觉得中国的玩家只能接受中国风格的游戏?照此分析,美国版的西游记又应该被改成什么样子?民族的才是世界的,这句话都被提的快烂掉了,难道在游戏橘子的眼里,这个”民族”就单单是中国的”民族”?其实大多数的玩家都是想玩到原滋原味的eq2,而不是这个该死的eq2东方版,对不起了,游戏橘子,这里请原谅我用了”该死”这两个字,不过现在其实也无所谓了,因为已经死掉了………….. 2,可能有人也会说,很多欧美经典游戏都失败的例子,但是我想只要玩游戏的,有几个人没玩过”大菠萝”没玩过”星际”? 又有几个人没玩过”cs”?还有很多很多例子.很多优秀的公司,都在中国取得了他们应有的成绩.为什么单单eq2中国玩家就会”不适应”呢?我想不适应的是游戏桔子你们的服务吧!本人也是第一批拿到测试号的人.试问当初恶劣的翻译,频繁的掉线.难道不是失败的原因?而且在后来的公开测试中,那令人啼笑皆非的翻译依然无处不在!试问这样的游戏环境,谁能接受? 3,宣传啊宣传,同样重量级的游戏,eq2和wow,你看看人家wow的宣传,再看看eq2的宣传!当然了,这个问题又归结到前面提到的了.其实光有好的宣传是不够的,宣传是可以带来人气,服务才是可以留住人气! 4,游戏配置,这个其实根本不是问题的问题.也只好提上一提了.因为心爱的eq2已经离开中国了.总得为它的离开,私人赞助一个理由了…………….. 最后我想说的就是,eq2是个好游戏,但是游戏橘子绝对不是个好代理,至少不是eq2的好代理.橘子啊,你让所有的中国eq爱好者失望了! PS:同样是写于两年前的了,到现在都为这款游戏的失败而惋惜.另外不知道那个让我梦寐以求的”神作”–UO(网络创世纪)真的会进入中国吗? 前几天在网上看到新闻说EA在中国找到了个UO的代理运营公司.希望不是仅仅的噱头. PS2:前几天魔兽的服务器也出现了连续一个多月的问题,而且是几乎所有的服务器都出现了,但是9城真的走过来了.很庆幸WOW暂时不会死于中国….
