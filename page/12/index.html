也说"技术"足球

发表于 2008-06-20 更新于 2019-07-25 分类于 体育

现在无论是所谓的足球专家也好,评论员也好,球迷也好.看起球来都喜欢说个”技术”.足球本来就是一门上升的艺术的体育运动,既然上升到”艺术”的高度了.其中必须得有技术含量,也就是所谓的”技术足球”.但是问题出来了,现在大多数人眼里的”技术”就是花哨的盘带过人,仿佛这就是技术足球了.呵呵,对不起真的要笑下,试问如果只讲盘带,当今足坛也就是葡萄牙,西班牙类的拉丁足球算是技术了.巴西都算不上,但是有意思的是为什么最后的胜利不是这些所谓的”技术足球”取得胜利呢?你别光强调运气,身体什么的.就拿荷兰和德国来说吧,一般的球迷,包括很多足球的评论员张口就说技术不行,好听点的叫技术加力量.但是德国为什么能成为欧洲之王呢?别忘记欧洲杯的3星不是盖的,光靠意志和身体,就能取得这么骄人的战绩? 其实大家往往忽略了一个概念,其实穿插,跑位,抢截,特别是漂亮的一脚传球也都是技术.特别是现在足球的特点,各个球队都很注重防守.有时候漂亮的一脚传球远比花哨的盘带过人更实用.其实也更需要”技术”!最后我要说的就是,我不是否认盘带过人等花哨的动作,只是咱们看球还是不能光盯着一点来看.足球里的技术不单单是漂亮的盘带过人,还有很多表现形式,别光看一点来评价任何一支球队. 他们都很技术,也许是评球的您,不懂技术!

暴雪发布官方暗黑II免CD补丁!

发表于 2008-06-19 更新于 2019-07-25 分类于 游戏

暴雪的敬业态度确实值得每一家游戏厂商学习。今年1月中旬，暴雪为《星际争霸》和资料片《母巢之战》发布了1.15.2版补丁，去掉了游戏过程中的CD验证机制，也就相当于一个官方免CD补丁。

现在，“大菠萝”《暗黑破坏神2》及其资料片《破坏之王》也获得了同样的待遇，更新了今天的1.12a版补丁后也不需要CD了。经过这么多年的洗礼，相信不少玩家的光盘都快支撑不住了，所以说暴雪还是相当体贴的。

如果用户当初安装游戏的时候选择了完全安装，并且不希望在游戏的同时插入CD光盘，那么只需把光盘上的所有.MPQ文件复制到游戏目录即可，大部分用户只需要复制D2Music.mpq或者D2xMusic.mpq，但Mac用户复制这些音乐文件之后还需要重命名为“Diablo II Music”或者“Diablo II Expansion Music”。

对于执行了非完全安装的玩家，那就需要重装一次游戏，当然这次要选择完全安装了。

除了免CD，新版补丁还修正了在Intel Mac平台上Rosetta和OpenGL模式之间的兼容性问题。

看eq2东方版的倒闭---失败的"橘子"

发表于 2007-03-30 更新于 2019-07-27 分类于 游戏

前几天在网上看到eq2东方版关闭运营的消息.并没有感觉很惊讶,因为在所有关心并喜欢这个游戏的广大fans心中都有数的.相信大多数fans都预感到这个结局了.但是为什么这么好的一款游戏,或者类似的几款优秀游戏都在大陆折戟沉沙,屡遭败绩呢?比如eq1,2. 魔剑.等等….. 不能单说一句水土不服就能解释得了的吧?因为在中间还伴随着魔兽世界(wow)这个成功的例子.本人觉得别的游戏不说,单说eq2在中国的几个失误地方

1,东方版本身就是失败.为什么要搞个东方版?这个想法我一直很不理解.游戏橘子难道觉得中国的玩家只能接受中国风格的游戏?照此分析,美国版的西游记又应该被改成什么样子?民族的才是世界的,这句话都被提的快烂掉了,难道在游戏橘子的眼里,这个”民族”就单单是中国的”民族”?其实大多数的玩家都是想玩到原滋原味的eq2,而不是这个该死的eq2东方版,对不起了,游戏橘子,这里请原谅我用了”该死”这两个字,不过现在其实也无所谓了,因为已经死掉了…………..

2,可能有人也会说,很多欧美经典游戏都失败的例子,但是我想只要玩游戏的,有几个人没玩过”大菠萝”没玩过”星际”? 又有几个人没玩过”cs”?还有很多很多例子.很多优秀的公司,都在中国取得了他们应有的成绩.为什么单单eq2中国玩家就会”不适应”呢?我想不适应的是游戏桔子你们的服务吧!本人也是第一批拿到测试号的人.试问当初恶劣的翻译,频繁的掉线.难道不是失败的原因?而且在后来的公开测试中,那令人啼笑皆非的翻译依然无处不在!试问这样的游戏环境,谁能接受?

3,宣传啊宣传,同样重量级的游戏,eq2和wow,你看看人家wow的宣传,再看看eq2的宣传!当然了,这个问题又归结到前面提到的了.其实光有好的宣传是不够的,宣传是可以带来人气,服务才是可以留住人气!

4,游戏配置,这个其实根本不是问题的问题.也只好提上一提了.因为心爱的eq2已经离开中国了.总得为它的离开,私人赞助一个理由了……………..

最后我想说的就是,eq2是个好游戏,但是游戏橘子绝对不是个好代理,至少不是eq2的好代理.橘子啊,你让所有的中国eq爱好者失望了! PS:同样是写于两年前的了,到现在都为这款游戏的失败而惋惜.另外不知道那个让我梦寐以求的”神作”–UO(网络创世纪)真的会进入中国吗? 前几天在网上看到新闻说EA在中国找到了个UO的代理运营公司.希望不是仅仅的噱头. PS2:前几天魔兽的服务器也出现了连续一个多月的问题,而且是几乎所有的服务器都出现了,但是9城真的走过来了.很庆幸WOW暂时不会死于中国….

Google vs Microsoft

发表于 2006-03-18 更新于 2019-07-25 分类于 互联网

这个题目实在有点太大了.以本人的这点见识,很难说到问题的关键.但是就从自己接触的一些来分析一下吧. 1,search.这个现在网络最热门的话题.记得以前有人说过,上网即下载.现在呢?也许可以说上网即搜索了.无论你想获得什么方面的知识和咨询.搜索引擎都可以提供给你了.当然了搜索引擎也有高下之分.目前来看,虽然msn也推出了测试版的搜索引擎,但是毕竟目前还是google的天下.google得1分

2,blog.要问时下最热门的话题,blog也应该算一个了吧?上到明星,下到平民,几乎都要在网上书写一番.且不说质量如何,至少先占个位置.如果以提供blog服务来说,目前肯定是msn领先了.现在msn提供的blog服务,可以说已经占到中国的半壁江山.有人可能要说google的blogspot.com了,但是这个中国的ip经常性的访问不了,本身就是个问题.这几天听说google提供的黑板报也关闭了服务.无可置疑的msn得1分

3,email.网上流行的联络方式.从提供的服务质量来说,msn和gmail应该是平手.但是msn提供的附加服务目前应该比gmail要多的多了.gmail唯一的容量,可能也会因为msn的livemail的普及而失去其目前的优势.且算打平

1:1 好了

4,TM.现在网络最流行的联系方式已经从email变成tm了.这方面msn和google相比依然无可置疑的是no.1.google推出的gtalk虽然功能很不错,插件开发的也很到位,但是致命伤就是用户没有普及.所以这回合依然是msn得分

1 5,desktop search.这个未来的发展方向应该也是很乐观的.因为现在是信息时代.海量的信息和软件吞噬着大家的硬盘,这时候,桌面搜索应运而生了.它为大家提供了一种方便快捷的本机检索方式.这这方面,就本人的试用情况来说,从反应速度来讲,其实两家差不多,但是具体的服务方面,google显然要比msn的桌面搜索更出色! google 得1分. 当然google还有很多优势项目,比如Google Analytics 网站分析.Sitemaps分析等等,但是不是大众都可以应用的项目,就不拿来比较了.而且,单纯的网络公司,有很多不确定性在里面.据说,呵呵,应该是比较可靠的消息,google也将要推出自己的操作系统了,是基于linux的.相信等到google的操作系统真的面世的时候,才真的会有一番大战.目前来讲还是Microsoft的天下! PS: 还是两年前的文章了,现在看来,起码还应该比较下微软拼音输入法和GOOGLE拼音输入法,呵呵.
