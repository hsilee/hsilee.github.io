在IE8里支付宝和工商银行的解决方法

发表于 2009-03-24 更新于 2019-07-25 分类于 数码电脑

呵呵,题目可能有点不通顺.反正就是在论坛看到好多朋友都说自己的在线银行功能和支付宝在IE8里无法使用. 临时赶制这个文章,希望能帮助有同样困惑的朋友. 首先选择IE8的”工具”-“internet选项”-“高级”

; 里面有个选项

,”启用内存保护帮助减少联机攻击”(呵呵,名称好长啊~) . 把这个前面的”对号”取消, 然后关闭IE8,再重新打开. 然后还是到IE8的”工具”菜单.里面有个”兼容性视图设置”

打开后,把

alipay.com 和

icbc.com.cn 添加到里面即可. 最后你就可以在IE8里面使用工商银行和支付宝的功能了,如果想添加别的银行,方式如上即可.

IE8使用心得

2, 当打开一个新选项卡的时候,可以看到有两个选项; a,使用

InPrivate 浏览

,这个是一个类似”安全浏览”的模式,即在该模式下浏览网页,不会在IE8中留下任何隐私信息,包括：Cookie、上网产生的临时文件、网页历史记录、窗体数据和密码、防仿冒网站缓存、地址栏和搜索自动完成记录、自动崩溃还原

(ACR)、文档对象模型

(DOM) 存储等.这个功能还是很人性的. b,使用加速器

,这个没太弄明白,大概我个人理解应该是当你浏览特定网页或者某段文字的时候,会出现一个快捷方式,方便你进一步了解.这个功能以后再研究吧…. 3, 搜索窗口也更加的人性和开放了,不再是原来的微软一家独大,呵呵. 你可以自己添加喜欢的搜索服务提供商.并且也加入了类似划词搜索的功能. 4,兼容性也提供了不少,如果发现一个网页在IE8下显示不正常,可以在地址栏的右边,选择使用”兼容性视图”. 5,浏览速度也提升了很多,我自己的机器上只有Firefox,chrome,和新装的IE8,我打开了10个常用网址测试,chrome速度最快,IE8第二,我现在使用的Firefox最慢. 最后总结 通过使用IE8,真的能感觉到微软在进步!起码让用户感受到了微软的诚意.其实我一直觉得,微软不是做不好,是别的软件没给他威胁而已. 当然了,我这里毕竟不是专业的测试环境,打开的网址可能也不那么科学,只是说下自己的使用感受.

黑莓手机购机测试全攻略

发表于 2009-03-22 更新于 2019-07-25 分类于 数码电脑

中国现在有很多的莓控,本人也是一个. 由于现在黑莓手机还没正式进入中国,现在市面上能见到的机器全部为非正常渠道的机器. 所以机器的质量及选购就成了一个困扰广大”莓友”的一个大问题.下面我就详细说下怎样对买到手的黑莓手机进行必要的检测. 1.插上SIM卡,内存卡,上电池,盖上后盖. 2.开机的过程观察屏幕,键盘灯,轨迹球等是否亮，在开机的漫长过程中，按下每一个不同的键位，看弹性是否足够。 3.开完机器后,不要急着去操作键盘先,找个比较亮点的地方,光线强的地方一般8系列键盘轨迹球会不亮,用手挡住感应灯是否正常,正常的话,挡住感应后键盘和轨迹球灯会亮起.(72系列连续按右下那个电源灯键，看键盘屏幕灯是否正常)。 4.按1234567890+按死ALT键不放+所有英文字,删除键,回车,测试基本的小键.然后按拨打电话键,功能键,返回键,挂机键是否正常. **5.按功能键移动到手机上的大桌面图,上下左右移动是否正常,滚轮的就上下移动就可以,利用慢的速度一步一

步移动,看是否会有过快,反跳前或跳后现象.然后按下轨迹球(滚轮),测试确定键是否正常,是否有弹性,利用轻微的力度按下是否出现没回弹性就已出现点击

的情况,键盘的也是如此.** **6.打开网络信号，看是否搜索信号正常，拨打10086测试通话,其间试听筒是否正常,在通话其间,按侧边

上下音量键是否正常，声音等级调至中等水平，用手捂住手机下巴位置到自己的嘴边(手势就是左手拿机,右手摆至左手),然后吹气,看是否能听见自己吹气声

音.也可以给朋友打电话测试。或录音测试送话是否正常。另外连接上耳机，测试耳机孔是否正常，耳机附件是否有损坏。** 7.在通话的时候,按免提键,升级型键盘71/81系列的按OP键,其他全键盘系列按回车键左边有一个＄符号的免提键，然后侧边按音量调至最大声，测试外放喇叭是否正常。 8.接着下来，就是测试左侧键，上顶键（除7290外）其他有上顶键左边的电源关闭待机键采取按死不放的方式，右边静音键在通话时按下，7系列测试侧边返回键。 **9.打开情景设置，测试震动。打开照相机功能，拍一张照片，开启闪光灯，然后手掌靠近摄像头再拍一张看是否

会使用闪光灯（也可以在黑的环境下试）。然后测试一次蓝牙开关闭，即打开之后，能否正常，关闭能否正常，如正常的，都能正常使用蓝牙功能，黑莓的接收和发

送蓝牙有点特别，具体的看论坛上的教程。** (1)GPS：找个空旷的地方，高级选项-GPS，刷新GPS，多刷几次看是否能搜索到星，经纬度。 (2)WIFI：寻找能用的WIFI网络，根据设置无错误之后，然后使用自带的浏览器是否正常上网。 10.进一步确定机器所有功能正常，选项（设置）–状态–输入test，调出测试模式进行测试. (1)测试休眠皮套，把机器放到休眠皮套去，看是否会自动熄灭屏幕，超级个别机器会因为自身主板的质量问题，出现无法休眠的情况。

(2)支持内存卡扩展的，插上内存卡开机之后，到多媒体查看是否能显示容量大小，如不能显示，格式化卡清理卡槽，如还是无法显示容量大小，先从内存卡原因去排除，再到硬件问题，不能读内存卡的有质量问题机是存在的。

最后在补充一些; 一.最后一步，使用充电器测试时候能够充电，手机能显示充电图标时，即代表充电正常，USB自然也是正常的，连接数据线使用DM肯定就百分百没问题。 二.能充电，就肯定能与电脑连接，首先安装桌面管理器是必须的，但遇到DM无法连接的情况，大多数你是没有在DM选项–连接设置检测PIN，多连接几次就OK。其次就考虑数据线是否损坏。 **三.另外很多人遇到该应用程序无法加载的现象，此种现象大多数是因为电脑的系统版本缺少某些补丁之类的东西。

此时，可尝试改用其他版本DM，把黑莓设置成英文系统，或进入应用程序权限，设置为默认。另外就是查看所需要安装的COD+ALX文件，是否放在同一目

录，或者使用ALX转换器，重新把COD转换出ALX文件，然后再次使用加载。（或在电脑重新刷一次手机，可以改善）** 以上三点，我从来没遇到过不能连接的情况，我这里都一直很正常，以上整个过程好像很罗唆，但写的明白点，详细点，大家也看清楚点，测试清楚点。 还有一点关系到8系列最重要的，87的不存在此问题。 1。查看选项目（设置）关于，查看版本是否为4.5版本。 2。如不是就立即测试刷机升级4.5版本ROM，如无法升级4.5更新出现失败的，恭喜你买到了有质量问题的机器。 3。立即采取刷回4.2ROM的方式，把机器刷好，退还给卖家。

吸血驴,一些不得不说的事

发表于 2009-03-16 更新于 2019-07-25 分类于 互联网

最近互联网上最争议的事,可能就是emule整个社区要封禁迅雷和vagaa.可能很多朋友正在使用迅雷和vagaa,但是却不清楚为什么会出现封禁. 让我们先来看看WIKI关于吸血驴的定义吧. 什么是吸血驴？？

吸血驴，也称吸血骡。是指在

eMule/eDonkey等一些P2P网络软件中通过其他用户下载完毕资源后而不再上传或只上传非常少的资源的用户。是由某些的商业公司或部分程序员对软件进行二次开发后的成果。吸血驴的出现被认为是违背了网络资源共享道德的产物，但使用者并不少。

通常，吸血驴（吸血骡）的行为被定义成“上传全是为了下载”而特别订制的特点都可以叫做吸血行为，它是违反共享精神的。用通俗的话来说，就是“私驴，自私的驴”。

好,现在就来简单说说为什么emule整个社区要封禁迅雷和vagaa. 在中国,vagaa以及迅雷使用用户非常多。在迅雷加载了emule模块之后，国内的迅雷吸血程度已到了让人不可容忍的地步了. 迅雷的emule模块首先不是遵循开源开发标准，也不符合emule社区关于emule mod的开发标准。

而且迅雷并不是只下载，不上传，它也上传，但是上传不是对全部emule使用者，他们的上传用户只是迅雷用户，也就是说迅雷实际上和vagaa是一丘之貉! 使用公平的emule端，你上传，获得积分，积分你无法看到，你只能看到你的上传下载比，积分能让你在他人那里排队时获得的号码更前，使你能更快的下载自己需要的部分。

封禁迅雷已经是没有办法的事情，就好像封禁vagaa，VC社区以及ed2000的热心会员，讨论了很久，并且由vagaa的事情，中国emule差点被整个emule社区档在门外，这件事情加速了封禁vagaa. 也许有人会说，emule太慢了，那么我告诉你，你只是想下载，不想共享，那么你永远都没有办法理解什么是共享精神.并且如果人人都这样想,最后的结果一定是所有人都没资源可下.

德塞公园

发表于 2009-03-15 更新于 2019-07-25 分类于 网站建设

让我们先来看一段真实的故事: 南非的德塞公园是在国际上招标建设的，中标的是一家德国设计院。当时就有非议。建成后，市民们更不满意，找出许多不尽如人意的地方。

后来南非人再建公园，就不用外国人了。20世纪70年代，南非人自己动手，修建了一个很大的公园—-克克娜公园。克克娜公园建好后，南非人一片叫好。它漂亮，气派，美丽。但两年后，南非人的看法却发生了惊人的变化。

在雨季到来时，克克娜公园被大水所淹，德塞公园却没有一点雨水的痕迹。德国人不但给整个公演建了下水设施，还垫高了两尺。这是当初人们不理解的地方，直到大水到来，人们才为此感到惊奇。

克克娜公园在举行集会时，秀丽的公园大门因为过小，而让人感到十分拥挤，甚至造成了安全事故。这时人们才大想到了德塞公园大门的过大给予了批评，认为它有点傻。

炎热的夏季，到克克娜公园的人们更为愤怒，因为它遮阳的地方太少，所谓的凉亭只是花架子，容纳不了多少人。而德塞公园纳凉的亭子，棚檐宽大，能容纳许多人。

几年后，克克娜公园的石板地磨损严重，不得不反修。而德塞公园的石板地却坚如磐石，雨后如新。而当初因为德塞公园的石板路投资过高，南非人差点叫德塞公司停工。当时的德国人非常固执，一定要坚持自己的做法，双方争得脸红脖子粗。当地人曾一度认为，德国人太死板、太愚笨。现在看来，德国人是对的。

还有草坪，南非人认为德塞公园的草坪过大，有点浪费，对此也隐隐不满。现在德塞公园的草坪面积却让人觉得正好，克克纳公园的草坪就显得小的多。

德国人在设计时，考虑到了南非的方方面面，包括天气与季节，地理与环境。南非人自己却没有顾及这些，他们竟然没有德国人熟悉他们生活的环境与状况。

德塞公园建完后，多少年没有变样，而克克娜公园总要修修补补，已经花掉了德塞公园两倍的钱。为此，南非同行感触甚多。有人曾经问德国同行，你们怎么会这么精明。德国人回答，我们只是实在，并非精明，精明的倒是你们南非人。

ps: 其实凡是实在的东西,在当初看来都有些平平淡淡.不足为奇.只有在经过了岁月和时间的洗礼后,才会显现出他的本来面目. 延伸开来,写BLOG是不是也是这样呢?很多人都在追求高PR值链接,SEO优化,关键词,静态化等等…… 其实要我说,只要你踏实的写上几年,你必然达到你的目的,可能不需要动那么多的脑筋. 平平淡淡真好!

move “%%i”

“%%~ni%%i”

) pause` 用上面的这个代码,保存成.BAT文件.在你要整理的文件夹里使用即可.如果你不会制作,可以用我提供的这个文件.

消除QQ网盘安全隐患

@echo off set /p id=请输入你的QQ号: rmdir “C:Documents and SettingsAll UsersApplication DataTencentQQDisk%id%temp”

/s /q pause 然后直接把这个批处理传到QQ网盘里

每次退出网盘前双击运行一下就可以了. 如果不喜欢自己编辑,可以使用我提供的这个dellQQdisk.bat,直接上传到QQ网盘即可. PS: 其实本人更喜欢用163和MSN的网盘,QQ现在太”向钱看”了.
