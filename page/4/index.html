bluehost空间申请免费SSH及搭建

发表于 2011-01-29 更新于 2019-07-27 分类于 网站建设

今天把前几天买的bluehost提供的SSH服务申请下来了.简单记录下,以备自己以后使用.

1,首先登录bluehost的cpanel管理面板,然后在help里搜索SSH.里面有个联系的链接,点进去. 大致的填写内容; “hi, Please enable SSH for my hosting account. The following is my account informaiton: primary domain: my-domain.com”

记得在附件位置上传一张自己的身份证照片. 然后把用户名和登录密码后4位填到指定的表格,还有你购买bluehost填写的email也一并填到指定位置,然后等待即可. 这里一定要表扬下bluehost,下面说明写的是在24-48小时内回复,实际时间5分钟.开通的邮件就发到我的邮箱里了.也可能是美国那边7点的原因吧.反正就是惊人的办事效率.

2,既然开通了,进去小小设置下,在cpanel面板-安全-SSH/壳访问

(点进去,生成自己的SSH公私密钥.这里设置很简单,起个名字弄个密码就完成了,然后把私人密钥下载到自己电脑即可)

3,SSH链接工具推荐用putty,毕竟是开源的,用着安全放心.网上还有一款中文界面的myentunnel.我没用:) putty配置简单说下,先在SSH目录下选择tunnels,目标设置为Dynamic,添加一个端口任意端口(比如8848 嘿嘿)，再按Add,一个动态转发端口就实现了. 然后用相应帐号ssh登录后,除了登录的终端窗口以外,刚才那个8848连服务器的22端口之间就有了一个SSH加密的转发通道了. 由于是ms-dos窗口

我就看到[~]就算是链接上了…. 有点简陋.不过下次再登录会有时间提示. 到这里SSH加密通道大功告成了.

4, 关于FTP工具上传,小小吐槽下啊.平时使用flashfxp顺手了,结果死活登录不进去…..后来老老实实用官方推荐的coreftp,好好享受了一把Sftp的快乐啊,速度确实比普通的速度快,而且还是加密的,呵呵

谁来保护我们的信息

发表于 2011-01-26 更新于 2019-07-25 分类于 无心呢喃

我是1996年开始上网的,应该算是中国第一代网民吧. 期间看过太多的网络公司起起落落.于是不由得的想到一个问题,谁来保护或者说保障我们的信息安全呢? 比如原来玩很多网络游戏,或者注册的很多信箱,还有很多即时联络工具等等,都是要求填真实的个人信息,现在这公司都石沉大海,渺无声息了.那设想一下吧,我们当初填写的个人信息,谁来保障不会流到外面,被不法的人所利用呢? 所以现在很多大型网络公司,facebook ,google,包括最喜欢山寨别人的疼训,现在都开始推广open ID.这本身是方便了我等上网大众,但是问题是当我们的资料被集中在几个公司的时候,会不会造成更大的不安全? 我所说的还只是网络的一小部分,只是关于我们隐私的冰山的一角,还有包括我们的手机,银行资料

,购房资料,等等等等……. 现在的中国已经进入了信息时代,是时候来考虑建立相关的法案,来保证我们的信息安全.如果只是从道德层面来规范,最后的结果就只能是造成全面的倒退. 因为不到一个月前的某微博关闭,随便说上两句,牢骚完毕,结束!

关于verycd关闭影视下载

发表于 2011-01-25 更新于 2019-07-25 分类于 互联网

我是23日知道verycd关闭影视下载的.这两天光忙于blog空间和域名的事,没来得及为它写点什么. 关于verycd的过去,我想不用过多介绍了.中国最优秀的影视资源提供站,这样的介绍一点不为过.特别在国内打击盗版影视,关闭众多BT站点以后,verycd更是广大影视爱好者的必去之地. 我不想从道义层面来评价verycd提供资源到底对不对.但是verycd的关闭也许真的宣告了国内免费下载影视作品时代的终结.VC的商业化,其实违背了emule的初衷.特别是去除了电驴（eDonkey）服务器和Kad网络的搜索之后,其实VC已经与当初真正的emule的共享精神,渐行渐远了.(注,eD2k服务器和Kad搜索都被取消了,easyemule只能访问VC的网站.由于这点,VC推出的easyemule被称为”阉驴”.) 虽然说现在还有几个PT论坛在提供资源,但是我想不远的将来,面临全盘整顿是必然的结果. 回想verycd陪我走过的这些年,只想说声感谢.毕竟在正版国外大片进入速度慢,又有数量限制的当今,verycd还是给我过很多快乐时光.曾经的vc必然要离我们远去,还是让我们怀着一颗感恩的心告别verycd吧!(补充下,原来也用过真正的emule.很多朋友纠结于VC的”阉割”eD2k服务器和KAD.但是就算把VC看作一个内容聚合来讲,毕竟也有它的功绩.但是毕竟还是去VC下载的时候最多.所以我个人还是要感谢VC) PS: 最具有讽刺的事,留在最后补充吧.呵呵,其实verycd有个山寨网站的,完全采集vc的所有数据.所以,现在它华丽转身,变成大家想下载vc资源的最好去处了

:) PS2:其实emule官方版本身可以搜索到所有的,包括VC提供的资源.但是不如在VC上看的那样直观,另外VC如果真关闭了,国家会不会接着整顿emule呢? 网址为

http://www.simplecd.org 要下什么资源的朋友速度出手吧,遭遇河蟹是必然的.

在bluehost安家了

发表于 2011-01-24 更新于 2019-07-27 分类于 网站建设

前几天国内的服务器工作人员提醒我了,由于我两个域名没有备案,必须停止解析.否则直接关停我的服务器. 由于自己其中3个域名是在国外注册的,死活也备不上案.无奈之下,就只能选择国外空间了.可怜我的域名啊PR值最多做到3.现在回归0了. 经过一番考量,最后还是选择了bluehost的主机,其实之前对godaddy印象最好的

.因为3个域名都是在godaddy注册的.但是相比较而言bluehost的主机都比较干净,没有色情或者政治类的网站.这样受牵连被”墙”的几率也能降低点.而且bluehost的主机最近还有折扣.呵呵3.95刀/月.真的便宜到家了.这里奉劝下,有像我一样想在bluehost安家的朋友们,尽量别相信什么代购的,有条件还是选择自己的信用卡支付最安全.如果害怕续费陷阱也可以选择用paypal支付.另外网上很多站都宣传只有点他们的链接才是3.95刀,经过我的实践,完全是胡扯. 想想自己这8年的独立blog路,真的很坎坷.第一个PR4的域名由于实在买不起服务器(这里真要BS下万网,那些年真太黑暗了,一个100M的静态空间都要几百大元),在05年掉了…. 07年注册,做到PR3的域名,因为服务器备案问题,又被停了3个多月,也降成1了.索性这次买空间直接就注册个崭新的域名

www.wxnn.net 和我的blog名字也很搭.正好是无心呢喃的拼音缩写,呵呵~~ 最后想说的是:新的一年,新的主机,新的域名,新的开始. 忘掉之前的艰难,重新上路!
