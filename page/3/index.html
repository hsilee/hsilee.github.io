windows 7 家庭普通版 启用Administrator超级管理员账户

发表于 2011-07-15 更新于 2019-07-25 分类于 数码电脑

一些OEM笔记本预装windows7系统也就是win7系统中常见比较低级的家庭普通版系统。

比如我现在用的就是联想的ome版的win7。

这个版本不知道微软出于什么考量，没有给用户使用超级管理账号的权限。

网上搜索的一些方法根本都不可用，经过研究发现其实开启超级管理员也很简单，只需要下面几个步骤即可；

1,在开始菜单中，搜索cmd.exe 点击鼠标右键，通过管理员方式运行。

2,在cmd中输入

net users administrator /active:yes 3,注销。

再登录的时候即可看到超级管理员账号了。

google的chrome浏览器版本(分支)说明

发表于 2011-02-22 更新于 2019-07-25 分类于 互联网

很多朋友都喜欢用chrome浏览器,其简洁的界面.独特的”沙盒”技术,快速的浏览速度为广大网友所喜爱,但是很多朋友下载的时候往往一头雾水,因为chrome的版本很多,下面我就为大家介绍下chrome各个版本都是什么意思. 1,Chromium Chromium的更新速度很快,每隔数小时即有新的开发版本.新功能会率先在Chromium上实现,待验证后才会应用在Chrome上，故Chrome的功能会相对落后但较稳定. 2,chrome Canary Canary分支要比Dev分支更要新,一些尚未进入Dev分支的新功能会先在Canary分支中出现. 3,Chrome dev 基于最新的Chromium Build,经常每周就更新释出的功能,与Beta版十分相似,但稳定性较差,不适合公共使用.基本是每周都有更新. 4,Chrome beta 基于Chrome dev,崩溃等重大故障较少发生,功能比dev更加完善.一般是每月更新. 5,Chrome stable 基于最新的测试版本，已知Bug均被修复，功能发挥稳定,一般是按季度更新. 综上所述,其实chromium和chrome可以理解成一个软件的Alpha和beta版本. 最后附上chromium和chrome的网址,均为Windows版本. chromium

活在春天里

发表于 2011-02-10 更新于 2019-07-25 分类于 无心呢喃

一把破木吉他,两个赤膊上阵的农民工,在一间狭小的出租屋里,尽情高歌,再加上旁边零落的啤酒瓶,嘴里的香烟.这已经构成了几乎全部的煽情要素.

“记得许多年前的春天,那时的我还没剪去长发,没有信用卡没有她,没有24小时热水的家,可当初的我是那么快乐,虽然只有一把破木吉他.”

旭日阳刚,两个创造奇迹的人.在短短的几个月中,演绎了中国传说中麻雀变凤凰的现代版.

其实我相信绝大多数善良的人们,都希望旭日阳刚能通过”春晚”这个阶梯,改善他们的生活质量,从此过上富足的生活.但是我同时又很矛盾,很怕他们真的富足了,也就不在是现在的”旭日阳刚”了.私以为大家之所以喜欢他们,同情是占了最大的成分.从某种程度来讲,他们的出现等于代表了广大的人民,特别是那些为生活所迫的草根阶层.人们之所以喜欢他们,其实也是因为他们完成了自己也许还没完成的梦想.但是当金钱的诱惑触手可及的时候,当他们不再代表广大的草根阶层的时候,还会有这么多的人喜欢他们吗?例如当年”憨憨的”王X强同学..

但是转念一想,其实自己的担心都是多余的.正如歌词里唱的那样,其实”旭日阳刚”已经活在他们人生的”春天里”.这已经足够了!

个人网站何去何从?

发表于 2011-02-05 更新于 2019-07-25 分类于 网站建设

中国人的过年都是差不多的套路,和亲朋好友聚聚,然后互相说点吉利话. 今天我喝酒了,可能说的和这个欢乐的节日有点冲突. 以前blog上也说过,自己应该算是中国互联网的第一代网民了.今天又重复,实在有点”倚老卖老”.因为看到了太多个人网站的兴衰.我原来在网上认识的个人网站的站长们,现在基本都已经不上网了.我不知道他们现在从事什么工作,是否还记得当初的梦想?问其实我现在对的互联网个人网站也感到很迷茫.我也不知道自己会坚持到什么时候. 看到很多新开的blog站长都在各种论坛里面推销着自己的网站.然后在自己的签名里也都加上自己的网站地址.其实能看出来,也都是怀着梦想的.我只想问大家一下,你真的点过别人的签名吗?也许在你的眼里,那些签名所代表的也就是个垃圾站,你根本都懒得一看的.但是现实其实是,反过来我们的网站在别人眼里其实也是一样的垃圾. 无论我们怀揣着怎么样美丽的梦想,现实总是那么的残酷.毕竟不是每个人都是技术高手或者作家又或者什么能吸引别人眼球的职业.那么当普通的网民实际搜索的时候,你觉得会在多少页以后才会看到我们的网站呢?这里先不说你网站的质量如何,排在前面的总会是那些给搜索引擎付费的网站,然后是各大门户站点,然后还有各种SEO团队等着呢…..然后,其实还有很多然后…….. 最后就算网民冲破重重的”障碍”来到了我们的网站,可能也就是看那么一眼,这一眼也许就是这个网民和这个网站的唯一一次”亲密接触”……. 其实绝大多数的个人网站可能都是互联网这个”汪洋”里的”一叶孤舟”.虽然这个描述很残酷,但却是事实. 其实我想说的是,每一个把自己的网站放到互联网上的朋友,都是怀揣着梦想的”毛毛虫”,只是最后能变成”美丽的蝴蝶”的几率实在是太低太低了. PS: 今天有点喝多了,回来无聊上网.习惯性的打开自己的网站.突然觉得很孤独,不知道自己在坚持什么,又或者想坚持什么,还能坚持到什么时候…. 又突然想到,自己写这么悲观的干什么呢?其实也是没人看的”可怜的个人网站”的一篇无聊文章而已. 算了,洗洗睡吧!
