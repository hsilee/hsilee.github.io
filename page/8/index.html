汉字输入法的思考

发表于 2009-04-14 更新于 2019-07-25 分类于 数码电脑

这两天在某论坛看到一篇热贴,是关于使用什么输入法,和打字速度的. 看完了,觉得里面很多人的回贴都很可笑.为什么呢. 很多”高人”都号称用某拼音输入法达到了每分钟160字以上的”惊人”输入速度. 天知道他们是怎么打出来的. 我这里简单说下现有的主流输入法,及这个”奇迹”现象的产生合理与否. 其实主流的输入法无非两种. 1,笔画

其代表是五笔输入法.记得在当年,这个是很多所谓的电脑学校,必备的学习课程.当然了,现在市面上还有一款”速录师”专用的输入法,这个需配合专用的键盘,使用群很小众. 2,拼音

拼音输入法就说来话长了.具体点就是全拼和双拼. 其中又派生很多拼音输入功能,比如首笔识别(智能abc为代表),声调识别(这个很多,就不列举了),和部首识别(自然码和智能狂拼.当然了,拼音加加和搜狗也带这个功能,但是需要通过附加键,比如”Tab”来激活,比较麻烦,主流还是自然码和智能狂拼).联想功能,这个是现在很多拼音输入法所宣传的,就是通过你输入一个句子,来智能判断你想输入的是什么. 输入法大概介绍了,下面说下打字速度是由哪些关键因素决定的. 1,重码率. 判断一个输入法的好坏,很大程度上是看”重码率”.比如同样的字母排列组合下,会产生几个汉字.这个决定了一个输入法的输入速度. 就重码率来说,五笔是完胜拼音输入法的. 2,击键率. 这个也是一个优秀的输入法的判断标准. 比如同样输入一篇文章,五笔类的任何一款输入法击键率肯定都是小于2的.也就是说平均每个汉字的只需敲击键盘不到两下即可. 而拼音输入法,能做到这点的,目前我接触只有自然码可以完美做到,我的45小时统计下,击键率为1.89.也不到2. 上面说了这么多,其实就是想说,正常的拼音输入法如果想达到论坛的那些”高人”所说的,一分钟平均160个字.其实是不可能完成的任务. 世界上的事肯定是没有绝对的,我也相信即使用全拼输入法.能有人达到160字/分钟的输入速度. 但是不能那么多人用某拼音输入法都达到这么惊人的速度. 重码率和每字击键率拼音输入法永远的痛.而且这里说的还没考虑南方人拼音不好的因素在里面. 最后想说的就是,如果你想从事现在高收入的速录师这个职业,还是乖乖的去买个专业的速录师输入法,只有这个能达到专业的要求. 如果你从事的是文案工作,平时的输入量很大,有时也需要兼职做个会议记录什么的.五笔才是你不二的选择. 如果你是从事创作工作(码字赚钱的),又或者你不想花时间去学习五笔,却想追求比较快的打字速度,那么我上面说的自然码就很适合你. 最后就是普通的拼音输入法了,相信我吧,无论他的广告说的多神,你学习的多刻苦,能达到真正的输入速度100字/分钟,你已经是输入神童了,而且你拥有这么高的输入天赋,用在别的输入法上,你的打字速度还会更上一层楼的. 就速度来说,普通的拼音输入法,永远是在汉字输入法这座金字塔的最底端.

开通yahoo信箱的POP3功能

发表于 2009-04-12 更新于 2019-07-25 分类于 互联网

Yahoo(雅虎)信箱一直以稳定著称,这也是很多用户喜欢他的原因. 但是yahoo信箱默认只能以在线的方式登录,他的POP3取信功能,需要手动在线开通. 首先登录你的雅虎信箱(据说Yahoo.cn是默认已经开通的,这个直接带POP3功能). 然后在浏览器的地址栏输入

http://edit.my.yahoo.com/config/set_popfwd?.src=ym&.done=http://cn.alonc.mail.com/ym/Options?YY=22797 就会进人到雅虎“网上直投”的页面. 在“注册”一项中选择“是”，然后选择你感兴趣的内容，在“选择频率”一项中选择“每周一至三次或每周一次”，其它的选项可根据自己的需要来选择，然后按“完成”按钮. 在出现的“选项”页面“步骤1－选择”中选择“网上阅读邮件和POP功能”，其它选项可根据需要选择，最后按“提交”按钮进人“SMTP配置”页面. 此时“网上直投”注册成功，系统会在此页面告诉你的“服务器设置”：

接收邮件（POP）服务器：pop.mail.yahoo.com.cn 发送邮件（SMTP）服务器：smtp.mail.yahoo.com(注意,网上有人说加上CN也可以收信,我这里测试没通过,如果这个不能发信,请使用smtp.mail.yahoo.com.cn) 其实Yahoo的意图很简单,如果你肯牺牲收取一些广告信,就可以免费使用他的POP3取信功能了.

不过是一个友情链接而已

本人也算上网很多年了.很多年的意思就是从96年算起,13年了. 在网上也见过很多稀奇古怪的事,帮过人,也被人帮过,受骗上当倒是很少.做BLOG是2002年.可惜后来间断了. 当时感觉周围的环境很好.大家有什么问题都互相交流,毫无保留.互相交换友情链接,就好比互相留了个名片. 从友情链接,你就看出这个BLOG主人的喜好,人以群分,物以类聚嘛.嘻嘻哈哈的都很快乐. 当然了,其中很多人现在都没有了联系,但是那份美好的感觉,是一辈子都不会忘记的. 这次重新写BLOG,其实没有了当初的惊喜,现在网上的任何东西,都变的很功利.就连个人的BLOG都一样. 想和谁交换个链接,就好比娶老婆,你得亮出你的财产(百度,Google搜录情况),地位(PR值是几). 当然了,这也无可厚非,当年的菜鸟不见了,现在都是SEO高手. 以上絮絮叨叨说了很多,其实是想说关于本人BLOG的友情链接的一点事情. 一直还是当年的思维,以为有几个友情链接,可以在网上不那么孤单.但是竟然遇到了”假友情链接”.具体什么是假的,大家可以去搜索下.我这里就不啰嗦了,网络也是个环境,到处都是copy,不利于”环保”. 这个事情给我的感觉就好想去享用一份丰盛的午餐,却发现午餐当中竟然有只苍蝇般的恶心. 这个人我已经从我的链接里去掉了,是谁自己最清楚.就算是”苍蝇”,我想也应该给他留点尊严. 最后提供两个查询友情链接的工具,希望和我一样单纯的朋友多个心思. 我们可以善良,但是我们绝不受骗! 第一个功能很强大,但是界面比较凌乱. 地址为: http://www.linkhelper.cn/ 第二个功能差点,但是界面好看,之所以提供两个,是为了避免”误伤”好人. 呵呵. 地址为:http://www.xianxingji.com/ PS: 我特意就这个事问过网上一个从事SEO的朋友,问他关于”假友情链接”有啥好处,他告诉我是可以提供排名,但是不影响PR值.

国内邮件推送(push mail)的思考

发表于 2009-04-02 更新于 2019-07-25 分类于 数码电脑

最近新买了个blackberry8220.经过一番折腾,进一步验证了黑莓的精髓就是push mail. 看了下移动提供的push mail服务,最便宜的都需要几百大元.实在是没这个必要. 但是又想尝试下”被推”的感觉,无奈之下,只好找找国内的已有解决方案. 1,berrymail提供的服务,说是免费的push mail,究其根本,其实还是pull.优点是让普通用户免费感受了下”假push mail”,虽然是”拉”,但是你可以骗自己是”推”啊.哈哈~~ 2, 尚邮据说是采用ip push方式,即为手机会保持一个固定的IP地址，邮件服务器收到新邮件后会在第一时间根据IP地址将邮件推送到手机客户端上.但是如果尚邮利用的是这样的ip push. 我个人的理解

当有新邮件到来时

, 绑定邮箱有邮件时先到达尚邮服务器上

,然后服务器会根据邮件的接收者信息.（也就是一对一的IP地址把邮件推送到客户的手机端上）.然后触发GPRS连接,来实现了邮件的推送.收到邮件后

断开GPRS连接

从而达到push mail的效果. 但是这样的话

为什么还有好多人诟病尚邮费电量费流量呢? 我看很多论坛上很多朋友都说尚邮其实是每隔一段时间

手机自己去服务器上拉（pull）邮件的.如果这样说就很好理解了.而且如果是这样”推”邮件,我不如直接开个POP3收信软件了. 尚邮的所谓push mail其实和berrymail的原理差不多. 说了这么多,感觉使用网上提供的免费服务不如利用gmail移动版自己收取了.起码信件安全有保证.于是又开始折腾gmail邮箱.首先让gmail可以自己收取我的hotmail,yahoo,qq信箱(关于收取hotmail信箱,可以查考我的日志”hotmail信箱的POP3设置“). 现在信件是都集中到gmail信箱里了,通过gmail移动客户端也可以收取.最后却发现,gmail的客户端是不能设置自动刷新的.天啊,想体验下假的push mail都这样难啊. PS:如果哪位朋友有好的方法,希望能提供下,真的很想被”推”一下.
