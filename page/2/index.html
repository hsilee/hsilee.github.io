修整博客

发表于 2013-09-14 更新于 2019-07-27 分类于 网站建设

今天正好比较清闲,把博客一直想动手修整,但是因为懒惰没动手的部分,都小小的修整一下.

1,图片

将本站原来挂掉的图片修补一下,这里要辩解下啊,因为这个博客是从2005年陆续写的文章,很多图片也找不到截图的环境了,总不能再重装一遍当时所有的软件或者相关的东西.感慨一下啊,时间过的真快,好多当年的”问题”,现在都早早解决了,比如那篇关于ie8的支付宝问题.比如关于当年的qq网盘监控文件,等等,都随着时间而改变了. 将能修补的图片重新整理到image文件夹,按照发布相关日志的时间,重新命名,以便以后备份修改等等.考虑到这博客几经波折(更换域名,更换空间,期间因为个人问题又停顿了半年多),现在也没什么流量了,所以图片也就没做防盗链设置.

2,友情链接

再一个整理就是友情链接,其实我很看重这个,不为seo,不为带来流量,只是觉得所有的交换过链接的网站,其实都是彼此的珍贵回忆.记得那时候最好玩的事,就是打开电脑,然后先看一下自己的网站,然后再去友情链接里的所有网站都去看上一遍.发现精彩的,再留个言交流一下.真是其乐无穷啊. 所以删除已经不在的网站,心情也很沉重,都是反复的去访问,甚至查看该网站的域名whois,只是怕错删一个还存在的博客. 3,删除垃圾评论

虽然装了akismet.但是还是有很多seo的评论,这次都删除了,我自己分析原来被”Q” 的域名就是受垃圾评论的牵连.由于全部为手动删除,肯定是有”误伤”了,在这里先跟被”误伤”的朋友道歉了.呵呵,其实也没人在意吧?

4,网站布局

将网站地图放到了博客的底部.本来这东西也不是给读者看的,都是迎接”爬虫”用的.放到上面显得布局很凌乱,我个人比较喜欢简洁

. 另外在底部加上赞助主机的链接,

5,日志

将原来日志里的错误链接修改了一些,比如指向我的旧域名的,又或者其它一些错误文字,等等. 6.robots和feed robots内容做了调整,另外把feed重新用feedsky烧录,feedsky很不靠谱,现已停用.

更换godaddy自助建站绑定的域名

发表于 2013-09-13 更新于 2019-07-25 分类于 网站建设

这个完全是写给自己的,网上相关教程都是关于虚拟主机的更换域名,对于自助建站更换域名,根本没有提及. 摸索了半天不得门路,后来在某论坛看到相关的信息了,赶紧记录下来,方便像我这样的新手设置. 登录websitetonight.com之后找到右上角有你注册名字的地方,然后点击,看到Domain setting ,选择Change Domain.然后修改即可,网址和英文攻略附上;

  1. Log in to your Website Builder account.
  2. Click Change domain.
  3. Select one of the following:
    • Select an available domain from your account —

Select a domain in your account from the Select Available Domain list.
* Enter a domain not registered in this account —

Enter a domain that isn’t in your account, such as one from another registrar.
* Enter a subdomain for a domain in this account —

Create a subdomain for your website, such as site.coolexample.com

  1. Click OK.

ps : 这自助建站本来是上当了.. 想注册一个心仪的域名,结果发现有1美金的虚拟空间.购买1年(12美金)即加送一个免费国际域名,贪便宜才去购买的.买来才发现是

website builder这种货. 一怒之下就随便绑了一个闲置域名,结果自娱自乐的胡乱搭建以后,发现这收费的空间就是不一样啊,打开网站以后,图片都是秒开的.于是才产生了想绑一个正式的域名,就算当作自己的个人展示站吧. ps2: 抢到的域名真不错,虽然是三拼米

,小小暗爽下,没枉费我这”挑灯夜战”的辛苦!

收了一个com的域名

发表于 2013-09-11 更新于 2019-07-27 分类于 互联网

现在4位字母,能拼出意义的,已经很少见到了.今天正好在网上闲逛,发现有人出售.价格又不是很贵.最巧还是双方都在godaddy.于是就出手了,在淘宝支付完以后,卖家在godaddy上直接push域名过来,前后也就几分钟.

说说顶级域名的后缀吧,本来com,net,org,biz,info都是顶级域名.但是在国内,现在占有率还是com一家独大.奇怪的是cn竟然比net和org更流行.这我也忍了,最受不了的是com.cn这朵奇葩,竟然如此受追捧.sina在推广com.cn上还是功劳大大的! 记得大概10年前的时候,做站的人很多喜欢用org的.据说搜索引擎会对org的后缀特殊优化,排名会靠前.至于info和biz,在中国就更可怜了.想想自己也还是随大流,弄个com比较靠谱. 闲话少絮,回归主题; 域名都买了,就拿出来晒晒吧, www.ylkl.com 自己想了几个含义,一路快乐,月亮哭了,要离开了,应力开裂,原来亏了,娱乐恐龙,原来开朗,哈哈.医疗什么的也行啊.还算比较灵活. 域名暂时也没什么用途,就先放着吧.再次提醒下自己,别再囤积域名了.浪费就是最大的可耻!

ps:重申下,最好的seo就是坚持写原创的东西.通过时间的积累自然而成.速成的东西都不靠谱,比如现在的饲料猪.

充电

发表于 2013-09-09 更新于 2019-07-27 分类于 数码电脑

今天给kindle充电,充了两个多小时,还没充满,又急着出门办事.于是想到了今天这个话题. 首先说说我自己的需要充电的东西吧

. 1,两部手机,这个肯定是无条件必须保证电量;

2,现在的智能机都很费电,为了保证在外面时候电量问题,特意买了一个移动电源,这玩意没事也得充电;

3,上面说的kindle,在任何时候,它都能陪我度过闲暇的时间,当然要充电;

4,平板,我个人觉得比笔记本使用率还要高,也要充电;

5,ipod,偶尔可以听听自己喜欢的音乐,要充电;

6,户外手电,需要使用18600电池,需要充电;

7,既然说到电池,就要说说我的G700鼠标了,这货简直就是在”吃”电池,两天一节,令人发指啊!无奈之下,10节eneloop伺候着,当然也要充电;

8,数码相机,家庭必备品,充电;

9,数码摄像机,必备品,充电;

10,笔记本,虽然携带时候比较少,但是一个月左右,电池也要充一次电; 11,蓝牙耳机,新交规下,必备品,要充电;

12,以上所说仅仅是我个人要充电的东西,可能还有遗漏.更不要说家庭为单位的话,起码要翻几倍….. 于是各种线缆,各种充电头,充电底座,车充,逆变器,等等…. 充斥着我的周围.不由的心生感慨,什么是伟大的发明,电池啊!如果谁能发明出一种真正的超高容量的电池,才算是造福现代的人类.另外就是各大厂商,各路神仙大姐,咱就不能坐到一起,把这该死的接口统一了?非得弄得搞出这么多的花样来不可. 牢骚完毕,结束

:)

网盘大战

发表于 2013-09-06 更新于 2019-07-27 分类于 互联网

最近互联网上最热门的消息莫过于网盘(或者叫云盘,比较时髦)的”战斗”了.大有你方唱罢他登场的气势. 简单梳理一下事情的起因和进程

: 首先登场的就是”搅屎棍”-360同学. 7月17日,360云盘率先提出1T空间赠送活动.这还就是个噱头,具体要很多条件的. 8月12日金山快盘跟进,推出免费100G活动.说两句闲话啊,金山现在越来越抠门啊,网盘速度也是大战的这几家速度最慢的一家.当年辉煌的金山已经没落. 8月14日360马上表示不服,提出送360G永久空间. 这期间华为也出来搅局,提出所谓的”无限空间”,不过也就是”无限外链”而已.话说网盘保存最多的就是各家的照片,文档等隐私文件,难道真用这个外链保存不成. 华为的思维首先就有问题,一笔带过,不提它了. 8月22日,这天应该着重说下,百度在”百度世界大会”期间推出1T网盘1元购活动.这个说真的,我挺震惊,还花2元钱充了两个账号

. 同天下午360同学马上表示不服,推出1T空间免费送,一分钱不花. 要不说丫的就是”搅屎棍”呢,各种不服啊. 8月28日好久没动静的腾讯终于行动了,有条件的将”微云”提升到10T (9月的时候又变成无条件10T空间了) …………..

到今天为止,各家目前的牌面是:

百度2T空间(需要花1元钱),360云盘是25级以下是36T空间.微云是10T空间. 华为还是老办法,无限外链. 草根网盘115,也终于行动了,采取的是乌龟式升级-每年增加1T. 上面把最近的”网盘大战”经过简单描述了一下.接着就说说我个人对这几家网盘的简单感受和未来发展的一个判断.

1,同步盘功能.(这样可能多台电脑,移动设备共享文件.) 其实这几家和老牌国外的相比技术方面差距还很是大(比如dropbox的连入设备自动备份上传功能,至今国内各路神仙没一个能做出来),但是国外的几家很吝啬,空间都太小了.. ;

2,速度,这是起码的,也是最根本的标准,不然你送我1万T,然后让我传它一万年,等于没送啊!通过我个人使用,觉得速度最好的竟然是”搅屎棍”360同学… 但是对360的以前各种”耍流氓”,心有余悸.还是选择了速度第二和第三的”微云”和”百度云”

; 3,抛开我个人喜欢的同步盘功能,国内的这几家其实是同步盘和资源盘混淆在一起的.所以也说说资源盘的情况吧,115和百度算是资源比较全的前两位了.起步早是很大的优势.华为的资源应该能排到第三位,再加上外链无限,稳坐三甲.至于微云和360云盘现在来说资源困乏是短时间无法解决的.只能通过时间来慢慢扳回局面了;

4,其实就目前国内蛋疼的网速来说,T盘已经足够使用,甚至有点多余.各家的容量之战应该告一段落了,下面真应该拼的其实就是稳定的服务和技术的提升,个人觉得最后赢家可能是在百度云和微云之间产生.百度依托于国内最大的搜索引擎,能利用搜索引擎给百度云带来无限的可能,比如人脸识别搜索,只要各位配合上传自己的照片到百度,这都是必然会发生的:) 微云的优势就是腾讯庞大的客户群.当然一切都是我个人的预测,结果真的怎样,让我们一起拭目以待吧!

windows 7 家庭普通版 启用Administrator超级管理员账户

发表于 2011-07-15 更新于 2019-07-25 分类于 数码电脑

一些OEM笔记本预装windows7系统也就是win7系统中常见比较低级的家庭普通版系统。

比如我现在用的就是联想的ome版的win7。

这个版本不知道微软出于什么考量，没有给用户使用超级管理账号的权限。

网上搜索的一些方法根本都不可用，经过研究发现其实开启超级管理员也很简单，只需要下面几个步骤即可；

1,在开始菜单中，搜索cmd.exe 点击鼠标右键，通过管理员方式运行。

2,在cmd中输入

net users administrator /active:yes 3,注销。

再登录的时候即可看到超级管理员账号了。
