小榕

发表于 2006-03-28 更新于 2019-07-25 分类于 无心呢喃

黑客像美国西部开发时的牛仔,没有法律的约束,但却有自己的做事准则。黑客要有道德底线,三条做黑客原则：不能仇视社会,不能给别人制造麻烦,不能给别人带来损失.小榕.呵呵,其实从未谋面,在网上也谈不到认识,只是曾经写信问过他些很”菜”的问题. 突然想起他,是因为最近不断发生的所谓”黑客”事件.一直以为,小榕才是中国为数不多的几个真正可以称得上”黑客”的人.永远记得小榕在自己主页说的话,”我不是黑客，因为我喜欢一切美好的东西，但是如果有什么破坏了这美好….Assault!无论在现实或是网络中，我都是孤独的…….”喜欢他给自己软件起的名字”朔雪”

“流光”……..很难想像一个理工科的人,能把软件的名字起的这样诗意且贴切. 记得在98年左右,自己也用”流光”做过一些令自己都很不齿的事.不过反过来也更证明了小榕软件的成功.呵呵,不管怎么样.中国应该多一些”小榕”.而不是现在的这些所谓”黑客”.怀念小榕.

Google vs Microsoft

发表于 2006-03-18 更新于 2019-07-25 分类于 互联网

这个题目实在有点太大了.以本人的这点见识,很难说到问题的关键.但是就从自己接触的一些来分析一下吧. 1,search.这个现在网络最热门的话题.记得以前有人说过,上网即下载.现在呢?也许可以说上网即搜索了.无论你想获得什么方面的知识和咨询.搜索引擎都可以提供给你了.当然了搜索引擎也有高下之分.目前来看,虽然msn也推出了测试版的搜索引擎,但是毕竟目前还是google的天下.google得1分

2,blog.要问时下最热门的话题,blog也应该算一个了吧?上到明星,下到平民,几乎都要在网上书写一番.且不说质量如何,至少先占个位置.如果以提供blog服务来说,目前肯定是msn领先了.现在msn提供的blog服务,可以说已经占到中国的半壁江山.有人可能要说google的blogspot.com了,但是这个中国的ip经常性的访问不了,本身就是个问题.这几天听说google提供的黑板报也关闭了服务.无可置疑的msn得1分

3,email.网上流行的联络方式.从提供的服务质量来说,msn和gmail应该是平手.但是msn提供的附加服务目前应该比gmail要多的多了.gmail唯一的容量,可能也会因为msn的livemail的普及而失去其目前的优势.且算打平

1:1 好了

4,TM.现在网络最流行的联系方式已经从email变成tm了.这方面msn和google相比依然无可置疑的是no.1.google推出的gtalk虽然功能很不错,插件开发的也很到位,但是致命伤就是用户没有普及.所以这回合依然是msn得分

1 5,desktop search.这个未来的发展方向应该也是很乐观的.因为现在是信息时代.海量的信息和软件吞噬着大家的硬盘,这时候,桌面搜索应运而生了.它为大家提供了一种方便快捷的本机检索方式.这这方面,就本人的试用情况来说,从反应速度来讲,其实两家差不多,但是具体的服务方面,google显然要比msn的桌面搜索更出色! google 得1分. 当然google还有很多优势项目,比如Google Analytics 网站分析.Sitemaps分析等等,但是不是大众都可以应用的项目,就不拿来比较了.而且,单纯的网络公司,有很多不确定性在里面.据说,呵呵,应该是比较可靠的消息,google也将要推出自己的操作系统了,是基于linux的.相信等到google的操作系统真的面世的时候,才真的会有一番大战.目前来讲还是Microsoft的天下! PS: 还是两年前的文章了,现在看来,起码还应该比较下微软拼音输入法和GOOGLE拼音输入法,呵呵.

firefox,ie杀手?

发表于 2006-03-17 更新于 2019-07-25 分类于 互联网

最近关于浏览器最流行的字眼莫过于firefox了.一时间仿佛大家都看到了浏览器的春天,甚至有人拿

firefox和ie的较量说成是当年ie和网景的翻版. 对于这样一款好评如潮的浏览器，当然要下来试用一下,我下的是最新版,Mozilla的

firefox简体中文1.501版.也下了几个插件.觉得FF的优点确实很多,开源,开放,同时也意味着更多的功能.但是如果想实现ie所有的效果,感觉内存使用上,首先要比ie大很多,cpu占有率也不如ie.当然这么直接比较有失公允,毕竟ie现在是和系统结合在一起的.但是firefox最大的商标是号称安全.可是真的安全吗?具体的编码就不讨论了,单就说普及率吧,记得在pchome的一个统计,通过firefox浏览该网站的用户连10%都没有占到,FF官方所谓的1秒几个人在下载我是相信的,但是具体试用后,决定继续使用的又有多少人呢?以上的数字也许正说明了FF的安全,其实是建立在用户少的基础上,用户少被关注程度也就会低很多.当然安全性也会高,但是既然大家都对FF的未来那么看好的,假设ie和FF的使用率颠倒一下的话,FF的安全性又能有多少?至少我觉得不可能达到ie的程度.再有就是ie的占有率决定了,很多网页都是only ie .这样的网页FF浏览的话,总有很多问题无法解决.因为具体涉及到版权的问题. 未来的FF究竟是ie的翻版还是,还是另一个昙花.大家拭目以待吧,至少我不认为firefox会真的成为ie杀手!

国际域名历险记

发表于 2006-03-03 更新于 2019-07-25 分类于 互联网

昨天和ken合计一下,是不是该注册个国际域名,也让自己的blog有个”国际名分”.于是我们决定注册一个.com的域名.上什么地方注册呢.网上一搜索,这域名价格差距真大啊.从150元的,到20多元的都有.选个便宜的吧.反正都是域名!ken提示去淘宝买,因为是电子支付么,那里相对保险.事后证明了她的英明!今天早上9点多,我的手机收到ken的短消息.说域名注册成功.呵呵,于是满怀欣喜的上网了.结果发现个问题.那个人(公司?)注册的域名,注册信箱是填的代理的,也就是他老人家自己的了.当时我们也没想太多,就是直觉告诉我们有点不对劲.我们的域名,为什么信箱是他的.后来就在网上查阅,最后才知道.原来这域名里的学问可多了.原来域名的归属依据主要就是联系人和注册信箱[这当然不行了.我们就和那个”域名注册公司”展开了拉锯式的讨论,期间还伴随无数个电话.我和ken的,ken和那个人的,咨询万网的.再加上QQ上的,历经一天的争论,直到下午,才算有个结果.就是答应给我们退款了.说到这里,真的要感谢世界上还有淘宝这个东西.可以事后付款啊,真是伟大至极!最后我们还是老老实实的去万网注册了个”贵”的域名.说贵,真的贵吗?那个便宜的陷阱少吗?注册信箱在他手里,一年以后的续费怎么计算且不说,这个域名到底是我们的还是他的,都说不清楚的!再加上n个长途电话,n+1个脑细.哎,真的是应了那句古话,”贪小便宜吃大亏”啊,古人诚不我欺!最后总结一下,注册域名,一定要找有信誉的大公司,别相信那些网上的”家庭作坊”(说好听点是sohu一族). ps:仅以此文纪念ggey.com诞生.

为了BLOG而BLOG?

发表于 2006-02-14 更新于 2019-07-25 分类于 互联网

记得在中国大多数人还不知道blog是什么的时候.弄了一个blog.还要在页首弄个关于blog的解释:)那时候个人blog很少,好像在google搜索还挺靠前,pr值是4. 后来域名没续费,blog自然也就不了了之了.最近偶然得到一个空间,又来了兴致,再想建立blog,才发现,现在的中国,blog已经好比大白菜一样的普及了.于是反思自己,为了什么要建立blog?思考了很久也没有答案,于是告诉自己,也许为了blog而blog. 是为第一篇网志. PS: 从这篇开始的日志都是我原来写在自己服务器空间的资料,现在空间没有了,只好都放到blogger上,现在又从blogger转到服务器上….生活在于折腾啊! 希望这次真的是永远的安家.(^__^)
