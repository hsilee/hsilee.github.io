iPhone使用OTG方式更新iOS10变砖解决方法

发表于 2016-09-15 更新于 2019-07-25 分类于 数码电脑

iOS10正式版。必须截图纪念一下，简直太惊险了。我一直很相信Apple的，所以每次版本更新都是直接选择OTG方式直升的，这次iOS10也不例外。结果给我来个变砖，折腾好久也无法启机。连到MacBook上用iTunes也无法恢复，一直提示无法连接更新服务器。

期间试过各种方法，断开重连、重启MacBook和iPhone、修改MacBook的DNS、V*P*N *fan*墙、修改本机Hosts文件、去官网下载iOS10正式版固件等等。都一样的提示，无法连接更新服务器！

更让人心碎的是，上次iPhone的备份是7月的，也就是说两个月的照片等资料完全都没有了。这样的结果肯定不能接受！

上网求助万能的Google。结果是一个自己都不相信的简单办法，完全解决了。

打开https://support.apple.com/en-us/HT203213这个网站，不要关闭，再用iPhone连接到电脑，打开iTunes。成功连接iTunes更新服务，恢复成功！

digitalocean搭建wordpress

发表于 2016-09-03 更新于 2019-07-25 分类于 网站建设

bluehost的虚拟主机用了6年多了。其实根本就没想更换，年纪大了也折腾不动。但是前几天备份BLOG的时候误删除文件了。和bluehost的客服沟通一周，倒是也回复邮件，但是一直在拖延，一周多了，还是没有恢复成功。

于是萌生更换主机的念头了。在google查找了一番，发现我完全落伍了，现在都是VPS的天下了，根本就没有人再用什么虚拟主机。想想也是花差不多的钱，干嘛要买虚拟主机？VPS本身还送一个独立IP，再加上无限的可操作性。于是就去digitalocean家购买了一个月的服务。买的最低配置的，5美金一个月。

结果实际一折腾，发现真的是“折腾”啊。一周下来，什么也没搞出来。于是就想到了网上的好友,老蔡同学。人与人的差距马上就体现出来了，我一周没搞定的问题。在他这里可以说很简单就搞定了。期间还教我好多linux的知识。

简单记录一下搭建过程，便于日后维护；

我选用的是CentOS 7 x64 主机。

一、安装apache；

1、sudo yum install httpd

点击y确认下载安装。

2、sudo service httpd start

开启web服务。

二、安装mysql；

1、sudo yum install mysql-server 安装。

2、sudo service mysqld start

开启mysql服务。

三、安装并搭建PHP环境；

1、yum install php php-mysql 安装php环境

2、测试php环境，建立一个探针

3、service httpd restart 重启apache服务器

四、安装wordpress 1、上传自己的wordpress安装文件（或者已有网站的全部文件，然后修改wordpress的config文件）

2、导入wordpres的相关mysql数据

以上就是我自己的大概搭建步骤。

新版Outlook for iOS/Android

发表于 2015-02-18 更新于 2019-07-25 分类于 数码电脑

这几天在试用Microsoft的新版Outlook for iOS/Android。感觉真的很不错，难得见到Microsoft能在移动端拿出这样有诚意的产品。支持市面上绝大多数信箱，163除外，不知道为什么163总是对非自家的APP各种不兼容（对iOS自带的mail支持也不好），可能出于推广自家APP的考量吧。说说新版Outlook for iOS/Android的大概优缺点（因为没有Windows手机，所以只说说自己在用的iOS/Android版）；

  1. 全平台支持，Android，iOS，Windows自家更不用说了，除了Linux，好像全包括了。
  2. 操作界面类似Mailbox，还可以设定滑动手势具体功能。
  3. 最喜欢的是移动版（Android，iOS测试过）里面的Gmail信箱自带翻墙，用着比较省心。（第一次认证时候需要翻墙，以后即可正常使用）虽然我的付费VPN还有2年期限，但是能减少步骤总是开心事。
  4. 支持iCloud信箱，对于iOS版可能无所谓，但是对于Android用户来说，绝对是一大便利啊。
  5. Android和iOS均支持push，而不是传统信箱APP的手动收取。而且电量控制很优秀。
  6. 唯一不喜欢的是强行读取Gmail的联系人，还找不到取消的地方，日历找到取消的地方了……这点不好，总要留给使用者选择的权利。
  7. 最后说一点缺点就是标记垃圾邮件功能，一直没有找到。总体感觉Outlook想做一款智能过滤邮件的APP，可能对于自己过滤垃圾邮件比较有信心吧。
  8. 很喜欢iOS版的的发送云盘附件功能（这功能在Android上就不是什么稀奇事，但是iOS上就算是便捷功能了。），支持国外主流云盘（Dropbox、Google Drive、OneDrive、Box全部集成）这个又提升了iOS上的使用效率。

因为iOS自带mail是不能发送附件的。

总体来说，瑕不掩瑜，可以说是Android和iOS上邮件管理的上佳选择。

安卓应用推荐（二）

发表于 2013-12-17 更新于 2019-07-25 分类于 数码电脑

前面一篇文章，介绍了一下自己对于安卓系统软件的推荐-安卓应用推荐（一），今天继续往下推荐，安卓应用推荐（二）新鲜出炉了。闲话少说直接开始；

1、淘宝移动版。尽管我不喜欢淘宝，里面充斥着各种假货。但是你不能否认的一点，只有淘宝上的货是最全的。所以也应该算是购物狂的必装应用了：）

2、支付宝钱包。作为淘宝的“好基友”。但是又不是仅此而已，支付宝现在就是国内移动支付的最佳选择，支持的银行最多，支持购物平台也是最多。

3、xposed框架。Xposed框架是一款可以在不修改APK的情况下影响程序运行（修改系统）的框架服务，基于它可以制作出许多功能强大的模块，且在功能不冲突的情况下同时运作。

当前，Per APP Setting（为每个应用设置单独的dpi或修改权限）、Cydia、XPrivacy（防止隐私泄露）、对原生Launcher替换图标等应用或功能均基于此框架。这个应用只能说是能力太大，祈祷别有人用来作恶了。由此衍生出来很多好玩的应用。

4、greenify绿色守护。此款软件的高级功能即是依附在xposed框架之下。另外还需要root后使用，这个软件不是三言两语能说明白的，简单说就是能让使用者自己选择安卓系统上的所有软件（包括系统软件）什么时候运行，而不是都在后台。从而提高安卓系统的运行速度。其实这款软件“所谓”的省电，更多的是一种“geek”的行为。虽然我本人是捐赠软件作者了。

5、autostarts：启动管理器。这个软件能分析你的系统有哪些程序是自动启动的。通过使用AutoStarts可以禁止系统在开机时把一些不常用的程序自动加载到内存中，从而达到手机启动后剩余内存比不用

AutoStarts设置之前大的目的。我是用这个和绿色守护搭配使用的。特别是最新版开始支持sony家的xperia系列了，表扬一下。

6、Nova Launche。4.0以上的安卓系统第三方桌面系统应用，桌面美化增强启动器类的双雄之一（另外一个是apex launcher）。简单说就是桌面应用增强美化。可以自定义手势（这样又可以少安装一个手势类的软件了），增加抽屉数量，动画效果等等。而且最值得称道的是nova 和apex的备份可以互通，这点绝对值得国内的软件作者去学习。

7、百度输入法。安卓上最好的输入法了。双拼，五笔，语音，笔画，网络备份词库。百度到底是做搜索引擎出身的，输入法方面就是有先天的优势，智能识别无可匹敌。这里要感慨下google输入法的不思进取，到现在双拼功能还没能实现，实在是很伤害像我这样的google fans的心！

8、百度地图。本来一直是用google地图的。但是百度的努力真的让国内的用户无法忽视。地图的正确率一直在提高，离线地图包更是国内昂贵的流量下最贴心的设定。唯一的缺点就是权限有点多，比如运行地图的时候，它会在后台偷偷把自家的应用都激活一下，比如贴吧，导航。另外就是没有google地图那么纯粹，现在开始弄什么雷达，购物推荐的，有点烦人，但是瑕不掩瑜，仍然是安卓上地图类的第一选择。

9、mx player。视频播放器最好的选择，没什么可说的，想看视频，就选它吧。

10、yahoo weather：雅虎天气。原来使用ios系统的时候，就觉得yahoo天气是最好的天气类软件，现在安卓系统也可以使用了。到底是国际大厂，制作精良，该有的功能全都具备，还没有国内同类软件的各种非必要权限，也没有可恨的广告。每天使用它查看天气的时候，都觉得是一种享受！

安卓应用推荐（一）

发表于 2013-12-05 更新于 2019-07-25 分类于 数码电脑

用安卓系统也3年多了，作为一个google fans，一开始就很关注安卓系统，不过原来一直用黑莓手机，直到3年前才入了第一款安卓手机－里程碑2。之后陆陆续续用过摩托的xoom平板，sony的两款，mt25i和今年的旗舰l36h。

我购买的都是国行机器，只能自己安装google服务包还有自行root。以下推荐的软件很多涉及到google服务还有root权限，先声明一下

：）另外本人极好折腾，所以各路软件都是安了删，删了安，也算是小有心得，以下推荐均为自己使用过而且和同类软件对比过的，算是给同好的一个小建议，也算是自己的一个安卓应用备份。

1、supersu。机器root后的最佳权限管理工具。补充一下，作者其实特意在设置的最后一项，可以让所有的使用者免费升级到专业版。所以不要去下载什么破解版，还不如在有能力的情况下，去直接赞助下软件作者。

2、RE管理器：root explorer。这个应该是安卓机root后，最好的系统文件管理工具。精简系统rom，瘦身的最好选择。新建文件夹，查看/编辑文件，软件安装，这些更是不在话下。有了RE其实就不在需要其它的文件管理软件了，新版还增加了搜索功能。

3、钛备份。想折腾安卓系统。这个应该是必须安装的，简单概括就是安卓系统上的“ghost”软件。其实功能还不仅仅限于备份恢复这么简单。（这里吐槽下google play商店。两次购买钛备份，均被拒绝了，现在彻底放弃了，转战amazon apps商店了，实在是伤害我的自尊心！）

4、luckypatcher：幸运破解器。既然说到了钛备份，就不能不提它的“好基友”－幸运破解器。这个绝对算上是必装软件了，钛备份牛吧？幸运破解器一样能搞定它。实际使用就不多介绍了，相信大家都会会心一笑，你们懂的！

5、gmail，凭良心说，这货就是email类软件的标杆，甚至ios的邮件系统也比不上gmail。更不用说国产的各类货色了。

6、亚马逊应用商店（amazon apps stroe），这个开始是被google逼的，购买无力，只能退而求其次，选择亚马逊了。不过确实没让我失望。方便的购买途径，再加上每周一款限免软件。缺点就是很多apps更新速度过慢，但是对于一个刚刚进入中国的服务，对待它还是要有一个宽容的心态。

7、chrome。在不考虑流量的情况下，chrome就是安卓上最好的浏览器。

8、微信。这玩意你还有啥说的，你不装它，你用啥？尽管它不遵循google的推送标准gcm，而是自己独立后台，尽管各种收费表情，各种脑残游戏。就算你说出它有一千个不如同类软件的地方，但是这货还真就是即时通讯软件的不二选择。

9、google search。有了这货，通讯录，网页，浏览器（貌似只支持chrome）的历史记录，所有的应用游戏，直接都能搜索到。再加上google now的卡片功能，安卓机的骄傲！

10、quickpic（快图浏览）。最好的多功能图片浏览器/图库/相册。软件本身小巧，无广告，而且也没有国产软件的通病－各种乱七八糟的权限。国产软件的良心之作，国产软件的骄傲，快图当之无愧！

试用BBM

发表于 2013-10-24 更新于 2019-07-25 分类于 数码电脑

这个“试用”只能添加引号了。因为不在是当年在blackberry上的那个BBM了。黑莓公司终于觉醒了，发布Android版和ios版的BBM了。个人觉得黑莓觉醒的太晚了点，而且还没开放非blackberry的语音和视讯功能。另外还需要申请排队，才能使用（虽然排队一天都会批准，但是还是会因为这个排队，流失很多想试用的客户）。

由于黑莓在中国的惨淡现状，也找不到什么联系人了。只能用自己的两部电话互相添加来体验了。不过正巧我的电话就是android和ios各一部，也算两个系统下都试用了。

简单说下体验吧；

1,由于非blackberry系统没有传统黑莓手机的pin码。所以只能用注册信箱的形式来使用。排队成功以后官方会给你一个pin码。

2,推送还是很犀利，不夸张的说，和短信比速度都是只快不慢。

3,上面说了，目前非blackberry系统只支持文字，语音和视讯等BBM的后续版本吧。

个人觉得BBM还是来的晚了，虽然当年引以为傲的推送还是很优秀，可是在现在的各种即时通讯软件包围之下，真的想不出BBM的生存空间在哪。还是以前说的，黑莓以死，只是BBer用来怀念而已。

调整win7的桌面图标大小

今天来分享一个win7系统下的小技巧，怎么调整win7桌面图标大小；

1,用鼠标左键选取任意图标。

2,选定以后，按住ctrl键，然后滚动鼠标滚轮即可。

3，鼠标滚轮向上是放大图标，反之是缩小。
