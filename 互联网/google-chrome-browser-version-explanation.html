google的chrome浏览器版本(分支)说明

发表于 2011-02-22 更新于 2019-07-25 分类于 互联网

很多朋友都喜欢用chrome浏览器,其简洁的界面.独特的”沙盒”技术,快速的浏览速度为广大网友所喜爱,但是很多朋友下载的时候往往一头雾水,因为chrome的版本很多,下面我就为大家介绍下chrome各个版本都是什么意思. 1,Chromium Chromium的更新速度很快,每隔数小时即有新的开发版本.新功能会率先在Chromium上实现,待验证后才会应用在Chrome上，故Chrome的功能会相对落后但较稳定. 2,chrome Canary Canary分支要比Dev分支更要新,一些尚未进入Dev分支的新功能会先在Canary分支中出现. 3,Chrome dev 基于最新的Chromium Build,经常每周就更新释出的功能,与Beta版十分相似,但稳定性较差,不适合公共使用.基本是每周都有更新. 4,Chrome beta 基于Chrome dev,崩溃等重大故障较少发生,功能比dev更加完善.一般是每月更新. 5,Chrome stable 基于最新的测试版本，已知Bug均被修复，功能发挥稳定,一般是按季度更新. 综上所述,其实chromium和chrome可以理解成一个软件的Alpha和beta版本. 最后附上chromium和chrome的网址,均为Windows版本. chromium
